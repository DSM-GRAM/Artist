# Artist
17회 대한민국 청소년 동아리 경진대회 출전을 위한 화가 게임

## App
Drawer, Shower, Scoreboard의 3개 앱으로 나누어 부스 구성을 준비.

- Drawer : 그림을 그리는 앱(스마트폰)
- Shower : 따라 그릴 샘플 그림을 보여주는 앱(Signage)
- Scoreboard : 랭킹을 보여주는 앱(Signage)
